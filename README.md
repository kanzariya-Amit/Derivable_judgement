# 📊 Health Data Analysis using Inferential Statistics

## 📌 Project Overview
This project focuses on applying **inferential statistical techniques** to a health dataset. The goal is to analyze various factors such as age, smoking habits, exercise, and medical conditions to derive meaningful insights and test hypotheses.

---

## 🎯 Objective
- Apply statistical methods to real-world health data  
- Perform hypothesis testing and interpret results  
- Understand relationships between variables using statistical techniques  

---

## 📂 Dataset Description
The dataset contains **500 records** of individuals with the following features:

- `record_id` – Unique ID  
- `age`, `age_group`  
- `weight`, `bmi`  
- `gender`, `region`  
- `smoking_status`, `exercise_frequency`  
- `blood_pressure`  
- `diabetes`, `hypertension`  
- `cholesterol_level`, `glucose_level`  
- `visit_date`  

---

## 🛠️ Technologies Used
- Python  
- Pandas  
- NumPy  
- SciPy  
- Matplotlib  

---

## 📊 Statistical Techniques Applied

### 1. Confidence Interval
- Estimated range for population parameters (e.g., age)

### 2. Z-Test
- Compared sample mean with population mean

### 3. T-Test
- One-sample and two-sample comparisons

### 4. Chi-Square Test
- Tested relationship between categorical variables  
  *(e.g., Smoking vs Diabetes)*

### 5. ANOVA Test
- Compared means across multiple groups  
  *(e.g., Age group vs BMI)*

### 6. Correlation & Covariance
- Measured relationships between numerical variables  
  *(e.g., Age vs BMI)*

---

## 📌 Hypotheses Example
- **H₀:** Smoking has no effect on diabetes  
- **H₁:** Smoking affects diabetes  

---

## 📈 Key Insights
- Statistical tests help determine whether observed differences are significant  
- Correlation shows relationships between variables like age and BMI  
- Hypothesis testing supports data-driven decision making
